# CutTime

CutTime เป็นโปรเจคเว็บแอปขนาดเล็กสำหรับบันทึกเวลางานของโปรเจคต่าง ๆ สร้างด้วย React และ Vite โดยมุ่งเน้นความเรียบง่ายและใช้งานง่าย

**คุณสมบัติหลัก**

- เพิ่ม/แก้ไขโปรเจค
- บันทึกเวลาที่ใช้ในแต่ละโปรเจค
- แสดงตารางโปรเจคและยอดเวลารวม
- โมดัลสำหรับเพิ่มรายการใหม่และแถบเครื่องมือเพื่อจัดการการแสดงผล

**เทคโนโลยีที่ใช้**

- React (JSX)
- Vite (dev server + build)
- CSS (ไฟล์สไตล์ใน `src`)

**เริ่มใช้งาน (Development)**
ติดตั้ง dependencies แล้วรันเซิร์ฟเวอร์พัฒนา:

```bash
npm install
npm run dev
```

การสร้างสำหรับการใช้งานจริง (production build):

```bash
npm run build
npm run preview
```

**โครงสร้างโปรเจค (สำคัญ)**

- src/
  - App.jsx — คอมโพเนนต์หลัก
  - main.jsx — entry ของแอป
  - components/
    - toolbar.jsx — แถบเครื่องมือ
    - project_table.jsx — ตารางแสดงโปรเจค
    - add_new_modal.jsx — ฟอร์มเพิ่มโปรเจค/เวลา
    - total_time.jsx — แสดงยอดเวลารวม
  - context/
    - ProjectContext.jsx — ต้นทางข้อมูล (state & provider)

**การใช้งานเบื้องต้น**

1. เปิดหน้าแอปในโหมดพัฒนา (`npm run dev`) แล้วเปิดเบราว์อร์ตาม URL ที่แสดง
2. ใช้ปุ่มบนแถบเครื่องมือเพื่อเพิ่มโปรเจคหรือบันทึกเวลา
3. ดูยอดเวลารวมด้านบน/ด้านล่างของหน้า

**ข้อเสนอแนะ / การพัฒนาเพิ่มเติม**

- เพิ่มการเก็บข้อมูลถาวร (localStorage หรือ backend API)
- เพิ่มการกรองและจัดกลุ่มโปรเจค
- เพิ่มการยืนยันการลบ และการแก้ไขรายการแบบละเอียด
